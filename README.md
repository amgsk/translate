# エンジニア翻訳

### 概要
プログラマーのための翻訳ツールです。  
日本語の文章から、変数名やメソッド名などの翻訳のお手伝いをします。

* 履歴は5件まで保存が可能で、古いものは自動的に削除されます。  
* 翻訳結果の単語にフォーカスすることで、単語それぞれの翻訳結果を閲覧することが可能です。
